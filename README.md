# webpack+Babel/ES6 backend scaffold

## Getting started

* Required:
    * [Node.js](http://nodejs.org): `brew install node` on OS X
    * [webpack](https://webpack.github.io): `npm install -g webpack`
    * App dependencies: `npm install`
* Run:
    * `npm run watch`
